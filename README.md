# Marketing Funnel Analysis Dashboard

This project analyzes marketing campaign performance using Power BI. It focuses on marketing funnel stages, conversion rates, platform performance, campaign effectiveness, regional insights, and revenue analysis.

## Features
- Marketing Funnel Analysis
- Conversion Rate Tracking
- Revenue by Platform
- Campaign Performance Analysis
- Regional Revenue Insights
- Target Audience Analysis
- Interactive Slicers
- Business Recommendations

## Tools Used
- Microsoft Power BI
- Microsoft Excel
- GitHub

## Project Outcome
The dashboard identifies conversion drop-offs, evaluates marketing channel performance, and provides actionable insights to improve lead-to-customer conversion and marketing ROI.
